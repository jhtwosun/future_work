## Abstract (verbatim)
We introduce a framework for calibrating machine learning models so that their predictions satisfy explicit, finite-sample statistical guarantees. Our calibration algorithms work with any underlying model and (unknown) data-generating distribution and do not require model refitting. The framework addresses, among other examples, false discovery rate control in multi-label classification, intersection-over-union control in instance segmentation, and the simultaneous control of the type-1 error of outlier detection and confidence set coverage in classification or regression. Our main insight is to reframe the risk-control problem as multiple hypothesis testing, enabling techniques and mathematical arguments different from those in the previous literature. We use the framework to provide new calibration methods for several core machine learning tasks, with detailed worked examples in computer vision and tabular medical data.

## Key idea (3-5 sentences)
Learn Then Test (LTT) reframes risk control as a multiple hypothesis testing problem: enumerate a (possibly multi-dimensional) grid Lambda of candidate hyperparameters, for each lambda in Lambda form a null H_lambda : R(lambda) > alpha, compute a finite-sample-valid p-value p_lambda from the calibration set, and return the set of lambdas that survive a family-wise error rate (FWER) controlling procedure (e.g., Bonferroni, fixed-sequence). Any returned lambda satisfies R(lambda) <= alpha with probability at least 1 - delta. Unlike CRC, LTT does not require the loss to be monotone in lambda, so it handles FDR, IoU, and other non-monotone risk surfaces; it is therefore the natural complement to CRC when the loss is non-monotone or when lambda is multi-dimensional. The price is reduced statistical efficiency relative to CRC when monotonicity does in fact hold.

## Method / theorem statements
- **Setup**: prediction algorithm parameterized by lambda in Lambda (finite or finite grid); risk R(lambda) = E[L_lambda(X, Y)] with L_lambda bounded; calibration sample of size n.
- **Per-lambda p-value**: for each lambda, construct a finite-sample-valid p-value p_lambda for the null H_lambda : R(lambda) > alpha (e.g., Hoeffding, Bentkus, or HBB inequalities applied to the empirical loss).
- **LTT algorithm**: run any FWER-controlling multiple-testing procedure (Bonferroni, fixed-sequence, sequential graphical) on {p_lambda}_{lambda in Lambda} at level delta; return the rejected set Lambda_hat.
- **Theorem (LTT validity, the result we cite)**: with probability at least 1 - delta over the calibration draw, sup_{lambda in Lambda_hat} R(lambda) <= alpha. Equivalently, every lambda in Lambda_hat is risk-valid simultaneously.
- **Assumptions**: i.i.d. (or exchangeable) calibration draw; bounded loss for valid concentration; finite or pre-specified search grid Lambda.
- **Slack terms**: deviation slack from the chosen concentration inequality (Hoeffding sqrt(log(1/delta) / n), Bentkus tighter for binomials); plus FWER multiplicity correction (Bonferroni log|Lambda|, or sharper for fixed-sequence).
- **Comparison to CRC**: LTT controls risk with probability 1 - delta (PAC-style), while CRC controls expected risk; LTT permits non-monotone L and multi-dimensional Lambda, while CRC needs monotonicity but pays no multiplicity tax.

## Reported results
- **Multi-label classification (FDR control)**: LTT delivers FDR <= alpha at the prescribed level on standard benchmarks (see paper for details).
- **Instance segmentation (IoU control)**: LTT bounds expected 1 - IoU on COCO-like setups (see paper for details).
- **Outlier detection + coverage**: simultaneous control of type-1 error and CP coverage validated empirically (see paper for details).
- All experiments confirm the predicted 1 - delta validity rate over repeated calibration splits.
